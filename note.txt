- Khởi tạo repository : git init

- Push code từ local len remote : 3 bước
+ Chọn : git add -A (git status: show thông tin)
+ Xác nhận : git commit -m "noi dung commit" , git log : check thong tin 
+ Đẩy : git remote add origin https://github.com/LeDinhSiPhu/demo_git.git (link GIT)
        lần đầu ~ git push -u origin master
        lần 2 trở lên ~ git push

*Branch (nhánh) 
Tạo nhánh: git checkout -b login 
Chuyển nhánh: git checkout master
